#  Real-Time Bus Arrival & Occupancy Information System (RTBIS)

A smart, IoT-enabled system designed to provide **accurate real-time bus arrival**, **location tracking**, and **occupancy information** to passengers using edge devices, cloud computing, and interactive audioâ€“visual displays.

---

##  Key Features
-  **Live GPS tracking** using NEO-6M module  
-  **ESP32 IoT node** for sending real-time data to the cloud  
-  **Firebase Realtime Database** for storing and updating bus data  
-  **Passenger Information Display** (Digital LED/LCD Screen)  
-  **Python + Tkinter GUI** with Google Maps integration  
-  **Edge processing** for distance + ETA calculations  
-  **Audio + visual alerts** for inclusive accessibility
